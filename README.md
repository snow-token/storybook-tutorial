# Storybook Tutorial

## はじめに

```
# パッケージのインストール
$ npm install

# Port 6006 でComponent Explore を起動
$ npm run storybook

# Port 3000 で ReactApp 起動
$ npm run start

# テストランナー起動
$ npm run test --watchAll

```

## Storybook の基本階層

Storybook には基本となる 2 つの階層がある。
ストーリーは `動作パターン` と呼び換えるとわかりやすいかも

- コンポーネント
  - ストーリー A
  - ストーリー B
  - ストーリー C
